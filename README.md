  - Adds thumbnail data inside MP3
  - Uses APIC tag (ID3v2)
  - Saves MP3 with embedded image

STEP 4: Clean Up

  - Deletes any .png files
  - Deletes any .webp files
  - Only MP3 remains!

STEP 5: Add Metadata
  - Adds song title

  - Thumbnail already embedded
